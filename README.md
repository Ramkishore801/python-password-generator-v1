🔐 Python Password Generator (Day 20)

📌 Overview

This is a simple password generator built using Python.
It generates random passwords using a combination of:

- Letters (A–Z, a–z)
- Numbers (0–9)
- Symbols (!@#$ etc.)

---

🚀 Features

- Generates random passwords
- Uses Python built-in modules ("random", "string")
- Combines letters, digits, and punctuation
- Beginner-friendly project

---

🛠️ Technologies Used

- Python 3

---

📂 How It Works

1. Import required modules
2. Combine character sets:
   - "ascii_letters"
   - "digits"
   - "punctuation"
3. Randomly select characters
4. Build password using a loop

---

▶️ Run the Program

python password_generator.py

---

📸 Example Output

gT@9#kL2!

🎯 Learning Outcome

- Usage of "string" module
- Looping and string building
- Basic logic building
